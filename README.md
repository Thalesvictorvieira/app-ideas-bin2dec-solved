# Bin2Dec - Decimal and Binary Converter

This project is a simple command-line application developed in Python that converts numbers between **decimal** and **binary** formats.

The idea for this project comes from the **Bin2Dec** challenge available in the
[florinpop17/app-ideas](https://github.com/florinpop17/app-ideas) repository.
